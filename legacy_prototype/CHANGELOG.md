# سجل التغييرات

## الإصدار 2.0 — تطوير شامل

### 🔐 الأمان
- إضافة `.gitignore` (حماية المفتاح السري وقاعدة البيانات والمرفوعات).
- تحصين الكوكي: `HttpOnly` + `SameSite=Lax` + `Secure` (اختياري).
- حماية CSRF عبر رأس `X-CSRF-Token` (مع تكامل الواجهة).
- رؤوس أمنية: CSP، X-Frame-Options، X-Content-Type-Options، Referrer-Policy، HSTS.
- وضع `debug` يُقرأ من البيئة (افتراضي معطّل).
- سياسة كلمات مرور (8 أحرف + مزيج أحرف/أرقام) + إجبار تغيير عند أول دخول.
- توحيد رسائل فشل الدخول (منع كشف الحسابات).
- تفويض على مستوى الملف في `/uploads` + خدمته كـ attachment.
- حدّ معدّل الطلبات (Rate limiting) لتسجيل الدخول وعامًا.
- مصادقة ثنائية (2FA / TOTP) اختيارية.
- خمول الجلسة (Idle timeout).

### 🐞 إصلاحات
- نقل الموظف ينقل كل السجلات التابعة (إصلاح تسريب العزل).
- تنفيذ حذف الموظف (الصلاحية كانت بلا وظيفة).
- CRUD كامل: حذف/تعديل للإقامات والخصومات والمستندات.
- عزل عدّاد الشركات في التقارير.
- تصحيح حساب الكبائس في EOS (365.25).
- تحويل الحقول الرقمية إلى أنواعها الصحيحة.
- التحقق من طول كلمة المرور عند إنشاء المستخدم.
- منع تعطيل آخر إدارة عليا أو الحساب الذاتي.
- معاملات (transactions) ذرّية للنقل والإنهاء.

### 🏗️ البنية والأداء
- وحدة إعدادات مركزية `config.py` (+ `.env`).
- فهارس أداء + تفعيل WAL + نظام هجرات (`PRAGMA user_version`).
- ترقيم صفحات للموظفين وسجل التدقيق.
- معالجات أخطاء JSON موحّدة + تسجيل (logging).
- خادم إنتاجي `waitress` (`wsgi.py`)، وتثبيت إصدارات المتطلبات.
- وحدات خدمة منفصلة: `payroll`, `leave_service`, `notify`, `mailer`, `sms`, `pdf_export`, `scheduler`.

### ✨ ميزات
- الأقسام والهيكل التنظيمي + التاريخ الوظيفي التلقائي.
- الحضور والانصراف + الساعات الإضافية.
- مسيّر الرواتب + قسائم الرواتب.
- رصيد الإجازة السنوية (احتساب + خصم).

- تصدير Excel و PDF (شهادة راتب، مخالصة).

### 🧪 اختبارات وDevOps
